#Vim Key Mapping

* `<c-f>` Search (using ack)
* `<leader>w` Save
* `:w!!` Save with sudo
* `<c-c>` Comment out/in
* `<leader>y` Copy to clipboard
* `<leader>p` Paste from clipboard

## Tabs
* `<s-l>` Switch right tab
* `<s-h>` Switch left tab

## Tmux
* `<c-l>` Switch right pane
* `<c-h>` Switch left pane
